1. Credential Check
`/api/login` 으로 아이디와 비밀번호를 전송합니다.
2. Refresh Token
응답 헤더의 `RefreshToken` 을 저장합니다.
3. Access Token
`/api/auth` 로 Authorization 헤더를 발급받습니다.
